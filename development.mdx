Flow Cloud is under active development. The docs distinguish between implemented behavior and planned behavior.

## Implemented today

- Authenticated Flow web app using Clerk.
- Projects dashboard and Flow IDE shell.
- Project file loading and editing through the project service.
- File version history and checkpoints.
- Vault-backed project secrets.
- Vault-backed environment variables with visible and editable values.
- Vault-backed certificates and provisioning profiles as write-only assets.
- Project Settings sections for General, Secrets, Environment Variables, Capabilities, Certificates & Profiles.
- Flow AI assistant rail that can generate code into project files.

## In progress or planned

- Production Mac build worker integration.
- Deployment and distribution flows.
- Full project export.
- Persisted capability configuration beyond generated entitlements.
- Richer API documentation generated from OpenAPI contracts.

<Warning>
  Do not document planned features as production-ready. Use "planned" or "not yet available" when describing incomplete behavior.
</Warning>
