Projects are the entry point to Flow Cloud.

## Project dashboard

The projects dashboard lists your cloud-native iOS projects. Opening a project loads the Flow IDE and initializes the project file tree.

## Active project recovery

Flow stores the active project ID and name in browser storage. This allows project-specific pages such as Project Settings to recover context after a hard refresh.

## Workspace menu

Use the workspace action menu in the IDE header to:

- Open **Project settings**.
- Return to **All projects**.

## Project identity

Project IDs are UUIDs. Services use the project ID when fetching files, listing Vault-backed configuration, and resolving build-time context.
