The Flow IDE is intentionally compact. It reserves the main screen for the active project task instead of showing global navigation chrome.

## Header

The IDE header contains:

- Flow brand and link back to projects.
- Workspace action menu for the current project.
- **Flow AI** toggle.
- Profile menu.

## Workspace switcher

The segmented control switches between:

- **Design**
- **Code**
- **Builds**
- **Preview**

The preview view hides the project file sidebar so preview can use the full workspace width.

## Action buttons

The action bar contains compact **Run** and **Deploy** controls. Export is not part of the IDE toolbar; it belongs in **Project Settings > General**.

## Right rail

Flow AI opens as a full-height right rail at the site-shell level. It is not nested inside the editor workspace, so it lines up with the entire IDE viewport.
