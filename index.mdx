Flow Cloud is a browser-native workspace for building and shipping iOS apps. It combines project storage, a web IDE, cloud build infrastructure, Vault-backed configuration, signing assets, and Flow AI into one connected workflow.

## What Flow gives you

<CardGroup cols={2}>
  <Card title="A real project workspace" icon="folder-tree" href="/ide/projects">
    Flow persists project files, metadata, file versions, and active project state so the IDE can recover after refreshes.
  </Card>
  <Card title="A browser IDE" icon="code" href="/ide/editor">
    Edit Swift and project files in Monaco, switch workspace views, create checkpoints, and inspect build output.
  </Card>
  <Card title="Vault-backed configuration" icon="key" href="/project-settings/secrets">
    Store secrets, environment variables, certificates, and provisioning profiles without exposing sensitive values in the browser.
  </Card>
  <Card title="Flow AI" icon="sparkles" href="/ide/flow-ai">
    Use the right-side assistant rail to generate code directly into project files with project context.
  </Card>
</CardGroup>
