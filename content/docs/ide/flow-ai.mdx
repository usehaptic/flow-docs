---
title: Flow AI
description: The project-aware assistant rail in the IDE.
---

Flow AI is the assistant experience inside the IDE. It appears as a full-height right rail and can stay open while the user works in the central workspace.

## Behavior

- Flow AI stays available while you work in the IDE.
- It uses project context to help with generation and edits.
- Generated changes can be applied directly to project files.

## Provider Keys

If Flow AI needs external model providers, users connect provider API keys from account settings.
