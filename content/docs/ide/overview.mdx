The Flow IDE is focused on the active project. It gives you dedicated space for editing, previewing, running builds, and using Flow AI.

## Shell

The IDE header contains:

- The compact Flow logo and wordmark.
- The workspace/project selector.
- Workspace tabs ordered as Design, Code, Builds, and Preview.
- Compact Run and Deploy actions.
- A profile menu.

## Workspace Tabs

Design, Code, Builds, and Preview are peer workspace modes. Preview hides the project/files sidebar to prioritize the running app surface.

## Assistant Rail

Flow AI opens as a full-height right rail. It is separate from the central workspace content so it can stay visible while the user switches workspace views.
