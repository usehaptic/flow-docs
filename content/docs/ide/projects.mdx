title: Projects
description: How projects are selected, loaded, and persisted in the IDE.
---

Projects are the unit of ownership for files, settings, build history, and outputs.

## Active Project

Flow remembers your active project so the IDE can restore the workspace after refresh. The active project is used when you open settings, edit files, run builds, and use Flow AI.

## Project Selector

The workspace selector is a compact action-menu style control. It shows the active project and provides access to project-level actions.

## Project Settings Entry

Project Settings is a dedicated authenticated page. It is not a modal, popover, or center-pane IDE view.
