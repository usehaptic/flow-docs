title: Design, Preview, and Builds
description: The non-code workspace surfaces inside the Flow IDE.
---

## Design

Design is the visual iteration space for project UI work.

## Preview

Preview prioritizes the app surface. When Preview is active, Flow hides the project/files sidebar so the preview can use more horizontal space.

## Builds

Builds shows build progress, output, and history. Run starts a build for the active project and selected environment.

## Deploy

Deploy is present as a compact IDE action. If deployment is not available for your project yet, Flow will keep the action disabled or show the current availability state.
