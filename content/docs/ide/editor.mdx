The Code workspace is where you edit Swift and project files.

## Layout

The editor fills the available workspace height and keeps the status bar pinned to the bottom. The project file tree is visible in Code mode and hidden in Preview mode.

## File Content

Flow opens files from your project and saves changes back to the workspace.

## Version History

The editor supports checkpoints and file version inspection so users can recover work during active development.
