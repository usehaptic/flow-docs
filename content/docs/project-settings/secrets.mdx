---
title: Secrets
description: Write-only sensitive values for a project.
---

Secrets store sensitive key-style project values. After a secret is saved, Flow does not show the value again in the UI.

## Environments

Secrets support these environments:

- Development
- Production

There is no Global or Preview environment in the current UI.

## Add Form

The add form is inline:

```text
Name | Value | Environment | Add
```

## Duplicate Handling

Secret names must be unique within the same environment. If you try to add a duplicate, Flow asks you to choose a different name or environment.

## Deletion

Deleting a secret requires confirmation before the value is removed.
